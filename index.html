Complexity Notation (Big-O)

Understanding how algorithms scale with input size:

O(1) O(log N) O(N) O(N²)

O(1) = constant time · O(log N) = logarithmic · O(N) = linear · O(N²) = quadratic

Base-12 Memory Addressing

Using base-12 as a locker system for memory access and organization.

Think of it like numbered lockers — each address points to a specific storage location.

Garbage Collection

Memory is deleted by removing all references to it when the system detects it is no longer needed.

Automatic cleanup — when nothing points to it anymore, it gets freed.
